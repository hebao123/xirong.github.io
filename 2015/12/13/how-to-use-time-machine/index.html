关于time machine不可不知的技巧

发表于 2015-12-13   |   分类于 mac   |  

Apple 官网上关于如何使用 time machine 的介绍，请看使用 Time Machine 备份或恢复 Mac ，下面的几篇文章都是手把手具体的介绍每个部分怎么使用的，可以阅读。

  1. Time Machine 使用教程（一）：设置 Time Machine 备份你的 Mac
  2. Time Machine 使用教程（二）：在 Time Machine 磁盘创建不同分区
  3. Time Machine 使用教程（三）：为你的备份加把锁
  4. Time Machine 使用教程（四）：排除特定文件（夹）的备份
  5. Time Machine 使用教程（五）：如何查看、恢复特定文件
  6. Time Machine 使用教程（六）：如何手动删除 Time Machine 备份
  7. Time Machine 使用教程（七）：使用 Time Machine 备份将音乐数据迁移至新 Mac

推荐排除下面的文件夹，一般没有什么用处，还可以减少备份盘占用空间的大小

  1. 用户/资源库/Caches这个文件夹不备份 里面主要是一些缓存文件夹比如我的里面就主要是帮助 safari Google earth的缓存文件。
  2. private/var文件夹下的内容不备份，里面就是系统休眠时把内存条里的内容转存到硬盘上存放的地方。
